# Project Name

A simple sign-up interface with subscription features and crypto-based transactions, built with React, Next.js, and Axios. This project will integrate Aptos Move contracts for secure crypto payments in future subscription models.

## Features

- User registration with form validation
- API integration using Axios
- Upcoming: Subscription model with Aptos Move contracts and cryptocurrency payment processing

## Tech Stack

- **Frontend**: React, Next.js, Tailwind CSS
- **Backend API**: Axios for HTTP requests
- **Blockchain**: Aptos Move (upcoming)
- **Crypto Integration**: Subscription payments via cryptocurrency (upcoming)
